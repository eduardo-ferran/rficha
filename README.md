# RFicha

RFicha is a cloud-based time tracking platform designed to help companies in Spain comply with mandatory working time registration regulations.

## Purpose
The platform allows employees to clock in and out digitally, while providing companies with auditable, legally compliant time records.

## Key principles
- Cloud-first, no mandatory local installations
- Immutable time records with full audit trail
- Multi-company and multi-department support
- GDPR-compliant data handling
- Inspection-ready reporting and exports

## Tech stack (initial)
- Frontend: Next.js (PWA + admin portal)
- Backend: API routes (Node.js)
- Database & Auth: Supabase (PostgreSQL)
- Hosting: Vercel
